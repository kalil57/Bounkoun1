# Bounkoun Core Service

This is the backend core service for Bounkoun.  
It handles:
- Project creation
- Topic suggestion + selection
- Research question generation + validation
- Workflow tracking
- Supabase database integration
- Anthropic AI integration

## Run locally
1. Create a .env file:
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
ANTHROPIC_API_KEY=

2. Install dependencies:
npm install

3. Start server:
npm start
